# Shop Api

1. clonar proyeco
2 . ```yarn install```
3. clonar el archivo ```.env.template``` y renombrar  a ```.env```
4. cambiar las variables de entorno
5. levanar la base de datos 
```
docker-compose up -d
```
6. correr el proyeco 
```
yarn start:dev
```
